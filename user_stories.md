# User Stories

## Admin User Stories
As an Admin, I want to manage access and system functionality to ensure the platform operates smoothly and securely.

- **Manage Users**: As an Admin, I want to create, update, and delete user accounts (doctors, staff) so that I can control who has access to the system.
- **Manage Doctors**: As an Admin, I want to add and update doctor profiles, including their specializations and details, so patients have accurate information.
- **System Monitoring**: As an Admin, I want to view a dashboard of system statistics (e.g., total appointments, active users) to monitor the platform's usage.
- **Manage Appointments**: As an Admin, I want to oversee all appointments and have the ability to intervene (e.g., cancel or reassign) in case of disputes or errors.

## Patient User Stories
As a Patient, I want to book and manage appointments easily to receive medical care.

- **View Doctors**: As a Patient, I want to search for and view a list of available doctors and their specializations.
- **Check Availability**: As a Patient, I want to see the available time slots for a specific doctor so I can choose a convenient time.
- **Book Appointment**: As a Patient, I want to book an appointment with a selected doctor for a specific time slot.
- **Manage Appointments**: As a Patient, I want to view my upcoming appointments and have the option to cancel or reschedule them if necessary.
- **View History**: As a Patient, I want to view my past appointment history and prescriptions.

## Doctor User Stories
As a Doctor, I want to manage my availability and patient appointments to provide efficient care.

- **Manage Availability**: As a Doctor, I want to set and update my available working hours and time slots so patients know when they can book me.
- **View Schedule**: As a Doctor, I want to view my daily or weekly schedule of appointments to plan my work accordingly.
- **Patient Details**: As a Doctor, I want to view details of the patient booked for a specific slot to prepare for the consultation.
- **Manage Appointments**: As a Doctor, I want to mark appointments as completed or add notes/prescriptions after the consultation.
